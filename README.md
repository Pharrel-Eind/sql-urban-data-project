# SQL Urban Data Project

This project simulates urban data using a relational database.

## Features
- ER diagram with citizens, buildings, energy, and transport
- 150+ records per entity generated using Python
- SQL queries for:
  - Energy usage analysis
  - Transportation per citizen
  - Empty buildings
  - Average energy per area

## Tools Used
- SQL
- Python

## Key Insights
- Identified high energy consumption patterns
- Analyzed citizen transport usage
